Extended Events Example

  • onbounce, onfinish, onstart: Watch marquee events fire. (IE4 Only)
    Bouncing message

  • ondragdrop: Try dragging a file onto the browser window. (N4 Only)
  • ondragstart: Try selecting text and dragging. (IE4 Only)

  • onselectstart: Try selecting this text. (IE4 Only)
